# Навыки (Skills)

Навыки - это модули, которые расширяют возможности бота. OpenClawGotchi поставляется с набором встроенных навыков, и ты можешь создавать свои!

## Встроенные навыки

### 🌤️ Weather
Показывает погоду для любого города.

```python
# Использование
/weather London
/weather Москва
```

### 📝 Notes
Создание и управление заметками.

```python
# Создание
/note Купить молоко

# Список
/notes

# Удаление
/delnote 1
```

### ⏰ Reminders
Напоминания по времени.

```python
# Напомнить через 5 минут
/remind 5 Перезвонить маме

# Напомнить в 15:30
/remind at 15:30 Встреча
```

### 🎵 Music
Управление музыкой (требуется настройка).

```python
/play
/pause
/next
```

### 📊 Stats
Статистика бота.

```python
/stats
/xp
/uptime
```
