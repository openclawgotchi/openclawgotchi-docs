🚀 Quick Start

Get your bot running in minutes:

  • Flash Raspberry Pi OS Lite to SD card
  • Clone the repository: git clone https://github.com/openclawgotchi/openclawgotchi.git
  • Run ./setup.sh — handles everything
  • Configure your Telegram bot token
  • Start chatting with your new AI companion!

🔧 Hardware

Designed for affordable, low-power hardware:

  • Computer: Raspberry Pi Zero 2 W
  • Display: Waveshare 2.13" E-Ink HAT
  • Storage: 16GB+ microSD card
  • Power: USB-C or micro USB power supply

🧠 Features

Your bot comes with powerful built-in capabilities:

Telegram Integration

Chat with your bot through Telegram. Send commands, ask questions, get updates. Full two-way communication.

Skills System

50+ skills available: weather, email, calendar, music, home automation, GitHub, Discord, and more. Easy to add your own.

Self-Awareness

The bot knows its own state, can modify its code, track its health, and even learn from interactions.

Memory System

Context-aware conversations with long-term memory. Remembers what you tell it, learns your preferences.

E-Ink Display

Shows mood, status, and custom messages on a low-power display. Always-on, always informative.

🛠️ Configuration

All settings in one place:

  • .env — API keys and tokens (never shared)
  • SOUL.md — Personality and behavior
  • IDENTITY.md — Bot name, owner info
  • MEMORY.md — Long-term memories

🤝 Brotherhood Mode

Connect multiple bots for inter-bot communication:

  • Mail system between sibling bots
